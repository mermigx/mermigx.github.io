TECHNOLOGY STACK


Scala

Scala is an hybrid language which combines the object-orented and functional paradigm and runs on the JVM.

Akka

Akka implements the Actors model on the JVM. It is a sound parallel programming model suitable for building highly concurrent, distributed, and fault tolerant event-driven applications.

Play Framework

Play is a stateless web framework to build highly-scalable applications with predictable and minimal resource consumption (CPU, memory, threads).

Ember.js

Ember.js is a MVC JavaScript framework to create scalable single-page applications. It provides an object model, declarative 2-way data binding, automatically-updating templates and a router.

Reactive Manifesto

The Reactive Manifesto describes an architecture that is event-driven, scalable, resilient and responsive: delivering responsive UX, using a scalable and resilient application stack, deployed on multicore and cloud.

Functional programming

Functional programming treats computation as the evaluation of mathematical functions and avoids state and mutable data. It is suitable for concurrent and parallel programming.

ASYNCHRONOUS WEB PROGRAMMING


People around the globe are accessing the web via their smartphones more often than ever. So far in 2013, 17.4% of web traffic has come through mobile. Smartphone users worldwide will reach 1.75 billion in 2014. They decide in as little as 1/20 of a second whether a site is appealing or not and they will visit a web site less often if it is slower than a close competitor by more than 250 milliseconds.

Remember when you were willing to wait a few seconds for a computer to respond to a click on a Web site or a tap on a keyboard? These days, even 400 milliseconds, literally the blink of an eye is too long.

Today’s web applications are integrating more concurrent real-time data, so web frameworks need to support a full asynchronous HTTP programming model.

The Actor-based model (Akka) is unquestionably the best model today to handle highly concurrent systems, and the best implementation of that model available for both Java and Scala is Akka - so it’s going in. Play 2 provides native Akka support for Play applications, making it possible to write highly-distributed systems.
